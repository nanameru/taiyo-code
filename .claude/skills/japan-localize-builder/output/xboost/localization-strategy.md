# XBoost（エックスブースト）ローカライズ戦略

### ビジネスモデル詳細
- **価格**: $29/月（月額課金）
- **フリーミアム**: 基本分析機能は無料、AI機能・スケジュール・高度分析はPro
- **課金フロー**: 登録→無料トライアル→Pro移行

### ユーザージャーニー
1. **登録**: Xアカウント連携でサインアップ
2. **オンボーディング**: 目標設定（フォロワー数目標、投稿頻度等）
3. **コア体験**: ダッシュボードで成長確認→バイラル投稿で学習→AI生成→スケジュール→分析で改善
4. **課金**: 無料制限に達したらProアップグレード
5. **リテンション**: 日次の成長レポートメール、週次ハイライト

### 成功要因
- **包括性**: 分析+AI+スケジュール+ネットワーキングを1つに統合
- **アルゴリズム理解**: シミュレーターという独自機能
- **クリエイターコミュニティ**: Social Hubによるネットワーク効果

## 2. 日本市場ギャップ分析

### 日本の類似サービスと弱点

| サービス名 | 機能 | 弱点 |
|-----------|------|------|
| SocialDog | 予約投稿、フォロワー分析 | AI機能なし、バイラル分析なし、UIが古い |
| Tweepie | リワード型ツイート | エンゲージメント分析が弱い、AI非搭載 |
| TweetDeck | マルチカラムUI | 分析機能が貧弱、AI非搭載 |
| keywordmap SNS | キーワード分析 | 投稿作成機能なし、高額（企業向け） |

### 日本ユーザーの未充足ニーズ
- **日本語に特化したAIコンテンツ生成**（英語LLMベースは日本語が不自然）
- **日本のバズ文化に特化した分析**（日本独特のバズパターン、大喜利文化等）
- **日本時間帯のゴールデンタイム分析**（通勤時間、昼休み、22時以降）
- **敬語/カジュアル切り替え**（ビジネス用途とプライベート用途）
- **日本のトレンドハッシュタグ分析**

### 文化的に変更が必要な点
- UIの情報密度を上げる（日本ユーザーは情報量を好む）
- 信頼シグナルの追加（利用者数、メディア掲載、運営情報）
- 丁寧な日本語コピーライティング
- 日本円での価格表示

### 日本ユーザーが求める追加機能
- 日本のトレンド/ハッシュタグランキング
- 日本語ツイートの感情分析
- カテゴリ別（ビジネス/エンタメ/テック等）のバズテンプレート
- 投稿カレンダーの祝日/イベント対応（日本の祝日）

## 3. ローカライズ方針

### UI/UX変更点
- **レイアウト**: ダークテーマ維持、情報密度を20%向上
- **フォント**: Noto Sans JP（メイン）、BIZ UDPGothic（コード/数値）
- **カラー**: ゴールド(#ffc367)をメインアクセントに維持、赤系(#e74c3c)をサブアクセントに
- **ナビゲーション**: 左サイドバー固定、アイコン+日本語ラベル

### コピーライティング方針
- **トーン**: フレンドリーだがプロフェッショナル（です/ます調）
- **敬語レベル**: B2C寄り、丁寧語ベース
- **キャッチコピー案**:
  1. 「Xの成長を、AIでブーストする」
  2. 「バズる投稿、AIが見つける」
  3. 「フォロワー1万人への最短ルート」
  4. 「あなたのXを、次のレベルへ」

### 決済手段優先順位
1. クレジットカード（Stripe）
2. (将来) PayPay / LINE Pay

### 日本独自機能
- 日本語AIコンテンツ生成（自然な日本語）
- 日本のトレンド分析
- 投稿カレンダーに日本の祝日自動反映
- 日本時間帯のゴールデンタイム提案

### 法的対応
- 特定商取引法に基づく表記
- プライバシーポリシー（個人情報保護法準拠）
- 利用規約
- インボイス制度対応
