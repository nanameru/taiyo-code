# サプリコ (SuppliCo) リサーチノート

## 調査日: 2026-02-24

## 元アプリ: Pillser (https://pillser.com/)

### サイト概要
- AIを活用したサプリメント分析・比較・推薦プラットフォーム
- 15,900件以上のサプリメントデータベース
- 4,400件以上の研究論文を索引化
- 7,400件以上の健康アウトカムを追跡

### 主要機能
1. **AI相談 (Pillser Ask)** — サプリメントに関する質問にAIが研究ベースで回答
2. **サプリメント検索・比較** — 成分ベースで検索、価格比較
3. **研究フィード** — 最新の研究論文を自動収集・表示
4. **パーソナライズ推薦** — 年齢・性別・体重・健康状態に基づく推薦
5. **ブランドディレクトリ** — ブランド別のサプリ一覧
6. **価格比較** — 複数サイトでの最安値表示

### UI/UX詳細
- **レイアウト**: クリーンなトップナビゲーション + 検索中心のLP
- **カラーパレット**: 白ベース + グリーン系アクセント（健康的なイメージ）
- **特徴的パターン**:
  - 統計数字の大型表示（15.9K Supplements等）
  - 検索ボックスが中心
  - カテゴリ別ブラウジング（Probiotics, Vitamins, Minerals）
  - 研究論文と直接リンクした信頼性表示

### ビジネスモデル
- **無料利用** — 基本機能はすべて無料
- **アフィリエイト収益** — サプリメント購入リンクからのコミッション
- **価格比較** — 最安値表示でアフィリエイト誘導

### 何が良い？
- AIと研究論文の組み合わせで信頼性が高い
- 無料で使える点が敷居を下げている
- サプリメント市場は巨大（グローバル$200B+）
- 検索→比較→購入の導線がスムーズ

### 何が微妙？
- デザインがやや地味
- パーソナライズ機能がもう少し前面に出せるはず
- コミュニティ機能がない
- 日本のサプリメント情報が皆無

### 日本向けに変えるべき点
1. **日本のサプリブランド対応** — DHC、ファンケル、大塚製薬、サントリー等
2. **日本語の研究論文対応** — 国立健康・栄養研究所等の情報活用
3. **「トクホ」「機能性表示食品」対応** — 日本独自の制度
4. **ドラッグストア価格比較** — マツモトキヨシ、ウエルシア等
5. **日本人の体質に合わせた推薦** — 日本人特有の栄養不足傾向
6. **よりモダンなUI** — 健康系アプリにふさわしい温かみのあるデザイン

## 日本市場分析

### 既存競合
- **キュアベル** — 市販薬・サプリ比較。管理栄養士コラム付き。UIは古め。
- **サプリメントデータベース** — ランキング形式の比較サイト。情報量は多いが使いにくい。
- **マイベスト** — 総合比較サイト。サプリ部門もあるが専門性に欠ける。
- **価格.com** — 価格比較。サプリ部門もあるが分析機能なし。
- **各社パーソナライズサプリ** — FUJIMI、VITANOTEなどは自社サプリ販売目的。中立でない。

### 競合の弱点
- AI分析がないか、あっても自社製品への誘導
- 研究論文ベースの客観的な情報提供がない
- パーソナライズが自社製品限定
- UIが古い、使いにくい
- 「トクホ」「機能性表示食品」の横断比較がない

### なぜ日本で成功できるか
1. **日本はサプリ大国** — 市場規模1.5兆円超、利用率50%以上
2. **AI×健康の組み合わせ** — 日本人のAI受容度は高い
3. **中立な情報源の欠如** — 既存サービスは自社製品への誘導が目的
4. **高齢化社会** — 健康への関心が年々増加
5. **機能性表示食品制度** — 日本独自の制度で情報ニーズが高い
