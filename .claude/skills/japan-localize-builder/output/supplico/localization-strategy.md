# サプリコ (SuppliCo) ローカライズ戦略ドキュメント

## 1. 元プロダクト完全分析

### 全機能リスト
| 機能 | 説明 | UI構成 |
|------|------|--------|
| AI相談 (Pillser Ask) | サプリに関する質問にAIが研究ベースで回答 | チャット形式 |
| サプリメント検索 | 15,900件のDB内を成分・カテゴリで検索 | 検索バー + フィルタ |
| 価格比較 | 複数ECサイトでの最安値表示 | 比較テーブル |
| 研究フィード | 最新の研究論文を自動収集 | カード形式のフィード |
| ブランドディレクトリ | ブランド別のサプリ一覧 | リスト + 詳細ページ |
| パーソナライズ推薦 | ユーザー属性に基づく推薦 | 質問フォーム → 結果 |
| サプリ詳細ページ | 成分、効果、研究エビデンス | タブ形式の詳細ページ |
| ユーザー質問 | コミュニティQ&A | Q&Aリスト |

### ビジネスモデル詳細
- **収益源**: アフィリエイト（サプリ購入リンク）
- **課金フロー**: なし（完全無料）
- **フリーミアム構成**: 無料のみ（広告なし）

### ユーザージャーニー
1. 検索エンジン/SNSから流入
2. 気になるサプリ/成分を検索
3. 研究エビデンスを確認
4. 価格比較で最安値を発見
5. アフィリエイトリンクから購入
6. 再訪問して新しい情報を探索

### 成功要因
- 無料で信頼性の高い情報源
- AIが研究論文を分かりやすく要約
- 中立的な立場（特定ブランドを推さない）
- SEOに強い構造（成分×効果のロングテール）

## 2. 日本市場ギャップ分析

### 日本の類似サービスと弱点
| サービス | 弱点 |
|---------|------|
| キュアベル | AI分析なし、UIが古い、更新頻度低い |
| サプリメントDB | 使いにくいUI、研究論文連携なし |
| マイベスト | 専門性不足、アフィリエイト寄りのランキング |
| 価格.com | 分析機能なし、価格のみ |
| FUJIMI/VITANOTE | 自社製品への誘導、中立でない |
| Dear-Natura | アサヒ製品のみ、他社比較なし |

### 日本ユーザーの未充足ニーズ
1. **「この成分、本当に効くの？」** — エビデンスベースの客観情報
2. **「トクホと機能性表示食品の違いは？」** — 制度の分かりやすい解説
3. **「DHCとファンケル、どっちがいい？」** — ブランド横断の中立比較
4. **「自分に何が足りない？」** — パーソナライズされた推薦
5. **「最安値はどこ？」** — ドラッグストア含む価格比較

### 文化的に変更が必要な点
- **敬語・丁寧語**: AIの回答は丁寧語で
- **エビデンスの重視**: 日本人は「科学的根拠」を重視する
- **安全性への配慮**: 副作用や飲み合わせ情報を強調
- **日本独自の制度**: トクホ、機能性表示食品、栄養機能食品

## 3. ローカライズ方針

### UI/UX変更点
- **レイアウト**: ナビゲーション + 検索中心は維持。情報密度を若干高めに
- **カラーパレット**: エメラルドグリーン (#10b981) + 白 + ライトグリーン（健康・自然のイメージ）
- **フォント**: Noto Sans JP（本文）、BIZ UDPGothic（見出し）
- **アイコン**: サプリメントの錠剤・カプセルをモチーフに

### コピーライティング方針
- **トーン**: 親しみやすい丁寧語（です・ます調）
- **敬語レベル**: 丁寧語（堅すぎず、くだけすぎず）
- **キャッチコピー案**:
  1. 「あなたに必要なサプリ、AIが見つけます」
  2. 「科学でわかる、サプリの本当の力」
  3. 「迷わない。比べて、選んで、健康に。」
  4. 「エビデンスで選ぶ、スマートなサプリ選び」

### 日本独自機能
1. **トクホ・機能性表示食品データベース** — 消費者庁データ連携
2. **日本のサプリブランド対応** — DHC、ファンケル、大塚製薬、サントリー、アサヒ等
3. **ドラッグストア価格比較** — マツキヨ、ウエルシア、ツルハ等の価格情報
4. **飲み合わせチェッカー** — 複数サプリの相互作用チェック
5. **季節別おすすめ** — 花粉症シーズン、夏バテ対策等

### 法的対応
- **特定商取引法に基づく表記**: 必須ページとして実装
- **プライバシーポリシー**: 個人情報保護法に準拠
- **免責事項**: 医療アドバイスではない旨の明記
- **アフィリエイト表示**: ステマ規制（景品表示法）に準拠
